## 页面结构

页面包括3个主要部分：
1. **项目管理** - 项目信息管理
2. **任务管理** - 项目计划管理和甘特图
3. **AI助手** - AI智能规划和问答 ⭐ NEW

### 2.3 任务列表操作说明

**操作列功能**：
- 新增子任务
- 编辑
- 删除
- 调整任务顺序（上下箭头），仅限同层级内

**工具栏功能**：
- **新增**：新增一级任务
- **保存**：保存项目计划到本地存储
- **导入**：导入项目计划（支持JSON、Markdown、Excel）
- **导出**：导出项目计划（支持JSON、Markdown、Excel）
- **导出甘特图**：导出甘特图为PNG
- **展开/折叠**：支持任务层级的折叠和展开
- **显示设置**：设置任务列表显示字段
- **AI智能规划**：启动AI智能规划功能 ⭐ NEW
